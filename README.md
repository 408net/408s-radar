# 408s-radar
408's Radar is a Python application that monitors Pokémon Retailers and sends Discord alerts into a server for events such as queue activation, product restocks and new product listings.

## Goals
- learn modern web scraping with Playwright
- monitor Pokémon Center for queue activation
- detect product restocks and stock changes
- track newly listed Pokémon TCG products
- send real-time Discord notifications
- build a scalable architecture that supports multiple retailers

## Roadmap
### Ver 1
- learn Playwright
- detect Pokémon Center queue
- send Discord notification
### Ver 2
- scrape one product
- detect stock changes
- store previous stock status in SQLite
### Ver 3
- scrape selected categories on Pokémon Center
- monitor multiple products
- detect newly listed products
### Future Ideas
- support multiple retailers
- local stock
- price history tracking
- user configurable alerts

## Status
Work in progress
